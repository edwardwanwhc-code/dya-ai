Hermes Agent 係咩？

Hermes Agent 係由 Nous Research（著名開源 AI 研究機構）開發嘅自我進化 AI Agent。 佢唔係普通嘅 Chatbot wrapper，而係一個真正嘅自主代理人：


✦ 自我學習 — 自動從使用經驗中創建 Skill，下次任務更快更準

✦ 跨平台存在 — 可以住在 Telegram、Discord、WhatsApp，唔係困喺個 App 度

✦ 60+ 內建工具 — 網頁搜尋、圖片生成、TTS、瀏覽器控制，一次過

✦ MCP 支援 — 連接任何 MCP 伺服器，無限延伸能力

✦ 支援本機模型 — Ollama、OpenRouter、Nous Portal 任你揀

📥 下載安裝

🪟 Windows
🍎 macOS

Windows 安裝（推薦：Desktop Installer）

1

下載 Desktop Installer

前往 hermes-agent.nousresearch.com 下載 Windows Desktop Installer，雙擊執行即可。安裝程式會自動處理所有依賴（Python、Node.js、ripgrep、ffmpeg）。

2

或用 PowerShell 命令安裝（進階）

PowerShell # 以 Administrator 身份執行 PowerShell，然後輸入： iex (irm https://hermes-agent.nousresearch.com/install.ps1)
3

完成後啟動

hermes # 直接對話 hermes desktop # 啟動桌面 GUI

macOS 安裝

1

方法一：Desktop Installer（最簡單）

前往 hermes-agent.nousresearch.com 下載 macOS DMG，開啟後拖入 Applications 即可。

2

方法二：Terminal 安裝

首先確保已安裝 Git（git --version 測試），然後執行：

Terminal curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
3

重載 Shell 並啟動

source ~/.zshrc hermes

Linux / WSL2 安裝

1

安裝前置需求

Bash # Debian / Ubuntu sudo apt install git curl xz-utils build-essential
2

執行安裝腳本

curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash

安裝程式會自動偵測環境，下載 Python 3.11、Node.js v22 等全部依賴。

3

重載並啟動

source ~/.bashrc hermes
4

（可選）啟動 Desktop GUI

hermes desktop

🔌 連接 AI 模型

安裝完成後，你需要連接一個 AI 模型提供商。以下是主要選擇：

🚀 Nous Portal（推薦）
一次 OAuth 登入搞掂，300+ 模型 + Tool Gateway（網搜、圖片生成、TTS）
hermes setup --portal
🔗 OpenRouter
按用量計費，支援 GPT-4o、Gemini、Claude 等主流模型
hermes config set OPENROUTER_API_KEY sk-or-...
🏠 Ollama（本機模型）
完全離線，隱私最高，需要本地 GPU 資源
# 在 config.yaml 設定 base_url: "http://localhost:11434/v1"
🤖 OpenAI / Anthropic
直接使用 API Key 連接官方 GPT-4o 或 Claude
hermes config set OPENAI_API_KEY sk-...

❓ 常見問題

安裝後出現 hermes: command not found 怎辦？ ▼
需要重載你的 Shell 設定檔。macOS/Linux 執行 source ~/.zshrc 或 source ~/.bashrc，Windows 重新開啟 PowerShell 即可。如仍未解決，執行 hermes doctor 診斷。
免費用得唔？需要俾錢嗎？ ▼
Hermes Agent 本身係免費開源的。但你需要連接一個 AI 模型提供商——Nous Portal 提供免費試用額度，OpenRouter 按用量計費，Ollama 本機模型則完全免費（但需要自備硬件）。
Hermes Desktop 同 CLI 有咩分別？ ▼
CLI（命令列）版係最核心的版本，功能齊全；Desktop 版提供圖形界面，更易上手。兩者功能基本相同，推薦新手先用 Desktop。
點樣更新到最新版本？ ▼
Hermes 會自動偵測安裝方式並顯示對應更新指令。通常執行 hermes update 即可。如有問題先執行 hermes config check 再 hermes config migrate。
可以喺 Telegram 用 Hermes 嗎？ ▼
可以！Hermes 支援 20+ 訊息平台，包括 Telegram、Discord、WhatsApp、WeChat、Feishu、DingTalk 等。執行 hermes gateway setup 進行設定。

- **Part 1** — Hermes Agent Desktop 完整安裝教學（繁體中文）

## 技術棧

- 純 HTML/CSS/JS，無依賴，直接 GitHub Pages 托管
- Mobile-friendly，深色科技風格

## 每週更新流程

1. 開啟 `index.html`
2. 找到 Part 2 的 `news-grid` section
3. 更新 4 個新聞卡片內容
4. 更新 `最後更新日期` 和期數
5. Git push → 自動部署
